# Crypto-Project-1
Password Manager project in Applied Crypto course

### Installation
- Clone this repo or download zip file
- cd to proj1_starter
```bash
pip install -r requirements.txt
```

### Run the test file
```bash
pytest -v test_password_manager.py
```
